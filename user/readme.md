# 📍 Mapa em Tempo Real

Projeto simples e funcional de rastreamento de localização em tempo real utilizando a API de Geolocalização do navegador e mapas interativos com Leaflet.js.

## 🚀 Funcionalidades

- Captura da localização do usuário em tempo real;
- Atualização contínua da posição utilizando `watchPosition`;
- Exibição da localização em mapa interativo;
- Centralização automática do mapa no usuário;
- Interface responsiva para dispositivos móveis;
- Integração com OpenStreetMap.

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Leaflet.js
- OpenStreetMap
- API Geolocation

---

## 📷 Preview

O sistema exibe:

- Um mapa interativo em tempo real;
- Um marcador indicando a posição atual do usuário;
- Atualização automática conforme o usuário se movimenta.


## 🌎 Como Funciona

O projeto utiliza:

- `navigator.geolocation.watchPosition()` para rastrear continuamente a localização;
- `Leaflet.js` para renderização do mapa;
- `OpenStreetMap` como provedor dos tiles do mapa.
